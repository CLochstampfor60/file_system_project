## CYSE 250 Milestone Project

# Secure File Sharing System - Project Design 

## Project Overview (Under Development)
A client-server application that allows users to securely upload, download, and manage files with encryption and authentication.

## System Architecture

### Server Component
- Listens for client connections on a specified port
- Manages user authentication (login/registration)
- Stores user credentials in a file (users.txt)
- Maintains file storage directory for uploaded files
- Encrypts/decrypts data during transmission
- Handles multiple operations: upload, download, list files, delete files

### Client Component
- Connects to server
- Provides user interface (menu system)
- Authenticates user with server
- Sends/receives encrypted data
- File upload/download operations
